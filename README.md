# Github viewer

This repo contains `webpack`, `AngularJS`, `ES6` and `Bootstrap 4`starter kit.

### Prerequisites
Node 6.x.x, Yarn

### Steps to run project
1. Clone Project.
2. Do `yarn install` to install all dependencies.
3. Finally `yarn start` to run project.
